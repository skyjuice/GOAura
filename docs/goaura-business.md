## 2. Market Size

```

## 3. Revenue Streams

GoAura operates **4 revenue streams** across B2C and B2B markets.

---

### 💳 Revenue Stream 1 — Interest on Microloans

The most direct and primary revenue source.

```
Loan amount:      RM 500 average
Interest rate:    18–24% per annum (standard micro-lending rate)
Loan tenure:      3–6 months

Profit per loan:
RM 500 × 18% ÷ 12 × 3 months = RM 22.50 per loan

At 100,000 active loans:
RM 22.50 × 100,000 = RM 2.25 million per quarter
```

---

### 🏦 Revenue Stream 2 — Scoring-as-a-Service (SaaS)

The Aura Score is not limited to TNG — it can be **sold to external institutions** as a standalone product.

**Who Would Buy GoAura Scores?**
- Banks wanting to reach gig worker segments
- Insurance companies pricing premiums
- BNPL providers (Atome, SPayLater, GrabPay Later)
- Telcos offering postpaid plans to unbanked users
- Landlords verifying tenant financial reliability

```
Pricing model:    RM 5–15 per score query
Volume:           50,000 queries/month

Revenue:          RM 250,000 – RM 750,000/month
```

> This is **pure margin** — the score is already computed. Selling it costs nothing extra.

---

### 🤝 Revenue Stream 3 — Financial Product Commission

When GoAura recommends a TNG product and the user activates it — **GoAura earns a referral commission.**

| Product Referred | Commission |
|---|---|
| CashLoan | RM 50–100 per approval |
| GOInvest | % of AUM (assets under management) |
| Insurance (GOprotect) | 10–15% of premium |
| TNG Visa Card | RM 30–50 per activation |
| GO+ onboarding | RM 5–10 per activation |

```
At 10,000 product activations/month:
10,000 × RM 50 average = RM 500,000/month
```

---

### ⭐ Revenue Stream 4 — Premium Score Subscription

Give users the option to **actively manage and improve** their Aura Score.

**GoAura Premium — RM 5–10/month includes:**
- Weekly score updates (free tier = monthly only)
- Personalized score improvement action plan
- Early access to new loan products
- Score simulator ("What if I save RM 100 more?")
- Priority appeal review for borderline scores

```
At 100,000 premium users × RM 7/month:
= RM 700,000/month recurring revenue
```

## 4. Customer Segments

GoAura serves **two distinct customer types:**

### 👤 B2C — End Users (TNG Users)

**What they want:**
- Access to credit they cannot get elsewhere
- A fair score that reflects their real behavior
- Financial products matched to their actual situation
- A clear path to improve their financial health

**What they pay:**
- Loan interest
- Optional premium subscription

---

### 🏢 B2B — Institutions

**What they want:**
- Access to the credit-invisible market segment
- Pre-scored, pre-profiled leads ready for product offers
- Reduced loan default risk through behavioral data
- Compliance-ready data pipeline

**What they pay:**
- Score query fees (per API call)
- Data licensing agreements
- API integration fees
- White-label GoAura scoring engine

## 5. Competitive Advantage

GoAura has **3 defensible moats** that are very difficult for competitors to replicate quickly.

---

### 🏰 Moat 1 — Data Exclusivity

```
GoAura is built on TNG behavioral data.
No competitor has access to:

→ 24M users' toll and RFID commute patterns
→ TNG merchant transaction history
→ GO+ savings behavior
→ LRT / MRT / bus transit data
→ Bill payment timeliness via TNG

Banks don't have this.
CTOS doesn't have this.
CCRIS doesn't have this.

Only TNG has this — and GoAura lives inside it.
```

---

### 🔄 Moat 2 — Network Effect

```
More users scored
        ↓
More repayment data collected
        ↓
Better, more accurate algorithm
        ↓
More lenders and institutions trust GoAura
        ↓
More financial products available to users
        ↓
More users join and engage
        ↓
Even more users scored
```

> The longer GoAura runs, **the smarter and more valuable it becomes** — automatically.

---

### 🔒 Moat 3 — Switching Cost

```
Once a user has an Aura Score history:
→ Their entire alternative credit identity is in GoAura
→ Starting over with a competitor means losing all score history
→ Lenders who trust GoAura are unlikely to switch platforms
→ Behavioral history cannot be replicated overnight
```

## 6. Risk & Mitigation

Every business carries risk. GoAura acknowledges these honestly.

| Risk | Impact | Mitigation |
|---|---|---|
| High loan default rate | High | Small first loans (RM 200–500), graduated limits |
| BNM regulatory changes | High | Built-in AML/CFT compliance, legal advisory |
| TNG data access restrictions | High | Formal partnership / licensing with TNG |
| Algorithm bias against certain groups | Medium | Fairness layers, context-aware scoring |
| User privacy concerns | Medium | Opt-in model, full PDPA compliance |
| Competitor replication | Medium | Data moat + network effect + 12-month head start |
| Foreigner flight risk | Medium | Malaysia-only loan usage, enhanced AML |
| Seasonal income misclassification | Low | Annual pattern analysis for seasonal workers |

## 7. Growth Roadmap

GoAura scales in **3 deliberate phases:**

### 🌱 Phase 1 — PROVE (Year 1)
```
Goal: Establish trust and validate the model

Actions:
→ Launch inside TNG ecosystem (pilot)
→ Score 1 million users
→ Offer microloans to top 10% Aura scorers only
→ Track and publish default rates
→ Prove that behavioral scoring = lower defaults than CTOS

Target metrics:
→ 1M users scored
→ Default rate < 5%
→ RM 50M loan book
```

---

### 🚀 Phase 2 — EXPAND (Year 2)
```
Goal: Grow revenue streams and user base

Actions:
→ Open Scoring-as-a-Service API to banks and BNPL providers
→ Launch GoAura Premium subscription
→ Expand foreigner segment (legal workers, expats)
→ Introduce blacklist recovery program formally
→ Integrate with GOInvest and GOprotect for commission

Target metrics:
→ 5M users scored
→ 20+ B2B score API clients
→ RM 200M loan book
→ RM 700,000/month subscription revenue
```

---

### 🌏 Phase 3 — DOMINATE (Year 3+)
```
Goal: Become Southeast Asia's alternative credit bureau

Actions:

Why regional expansion works:

Target metrics:
